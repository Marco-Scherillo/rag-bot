# rag-bot
RAG-based Q&amp;A bot — chat with your documents using Claude + ChromaDB

# How it works 
takes in an input such as .mk file or .txt file even .ppxt file embeds that then stores it in chromaDB does a query on AI, then AI answers.

# how do i run it? 
API key goes in the .env file, select an input then give file location.
